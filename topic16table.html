TABLE DESIGNING USING SCAMPER

We designed a table using SCAMPER which consists of elements like a wireless charging port , a display screen , led light strip, buttons for different functions ,a food heater, medical monitoring.

Here's a SCAMPER breakdown for the design of the table with multiple functionalities:

Substitute

Explore using alternative materials that are durable yet lightweight for the coffee table structure to accommodate additional features without compromising portability.

Combine

Integrate the wireless charging port and the display screen to create a multifunctional charging and entertainment hub. Temperature setter , medical monitoring ,heat setter all is connected to the digital screen which you can easily access according to your needs.

Adapt

Customize the LED light strip to change colors based on user preferences or ambient lighting conditions, enhancing the table's aesthetic appeal.

Modify

Added different levels to differentiate between the heater and working part.

Put to another use:

Utilize the food heater feature instead of additional inductions.

Eliminate

Mishappening of food items because of given spaces.

Rearrange:

Foldable table legs and a compartment for holding  beverages.
